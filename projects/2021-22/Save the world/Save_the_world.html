Description

SaveTheWorld is creative programming project, with the aim of raising the awareness of the impact of our gestures on the environment.

The goal of this project is to make people aware of how simple daily actions can heavily impact the environment. With this concept in our mind, the interactive installation sends a clear message to the user, i.e. that everybody can contribute to change things for the better, and so to "Save the World".
